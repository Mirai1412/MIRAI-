깃 연동
git init // 깃파일만들고
git remote add origin 주소 // 레파지토리 연동
git remote remove origin // 연동 끊기
git pull origin master // 안에 파일들 받아옴

   -- Traking 순서 --
깃 푸쉬
git status 상태확인
git add . 전부추가
git commit -m "message" 메세지넣고
git push origin master 전부 넣기
git push origin +master 강제 푸쉬

git branch 파일이름 마킹
git checkout 파일이름 마킹한놈 변경
git merge 파일합체

git rebase 전에 수정한것들을 볼수 있다.

rm -r .git을 입력하면 .git 폴더가 삭제되며 
Git 로컬 저장소 지정을 해제한다.
